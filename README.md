# 🤝 InAmigos Foundation — Awareness Website

A responsive, modern awareness webpage for **InAmigos Foundation**, a Section 8 registered non-profit organization working across 28 Indian states on hunger relief, education, women empowerment, animal welfare, environmental sustainability, and youth skill development.

## 🌐 Live Demo

> Deployed URL will appear here after hosting on Render / GitHub Pages.

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| **HTML5** | Page structure and semantics |
| **CSS3** | Styling, animations, responsive layout |
| **Vanilla JavaScript** | Slideshow, scroll effects, form handler |
| **Google Fonts** | Outfit (headings) + Inter (body) |

> No frameworks, no dependencies, no build step required.

## ✨ Features

- **Hero Slideshow** — Auto-rotating banner using official NGO images
- **Responsive Design** — Works on mobile, tablet, and desktop
- **Scroll Reveal** — Cards animate in as the user scrolls
- **Active Nav Highlight** — Navigation link highlights based on current section
- **Smooth Scrolling** — Anchor links scroll smoothly to sections
- **Contact Form** — Opens user's email client via `mailto:` with pre-filled content
- **Sticky Navbar** — Adds shadow on scroll for depth
- **Mobile Hamburger Menu** — Collapsible nav for small screens
- **Official Assets** — Logo and images sourced directly from [inamigosfoundation.org.in](https://inamigosfoundation.org.in)

## 🚀 Deployment (Render)

### Prerequisites
- A [GitHub](https://github.com) account
- A [Render](https://render.com) account

### Steps

**1. Push to GitHub**
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/inamigos-ngo.git
git branch -M main
git push -u origin main
```

**2. Create Static Site on Render**
1. Log in to [render.com](https://render.com)
2. Click **New +** → **Static Site**
3. Connect GitHub and select your repository
4. Configure:
   - **Build Command**: *(leave empty)*
   - **Publish Directory**: `.`
5. Click **Create Static Site**

Your site will be live at `https://inamigos-ngo.onrender.com` within a minute.
